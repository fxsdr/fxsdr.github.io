CARA GANTI FOTO
================
Taruh file foto Anda di folder ini (assets/images/) dengan NAMA FILE PERSIS seperti di bawah.
Format yang didukung: .jpg, .jpeg, .png, .webp (jika pakai format lain, ganti juga
ekstensi di file index.html pada bagian <img src="assets/images/...">).

1. couple.jpg   -> Foto pasangan di halaman sampul (cover)
2. adjie.jpg    -> Foto mempelai pria (Adjie)
3. galuh.jpg    -> Foto mempelai wanita (Galuh)
4. journey.jpg  -> Foto kenangan di bagian "Our Journey"
5. closing.jpg  -> Foto penutup di bagian akhir undangan

Selama file belum ditambahkan / nama tidak sesuai, halaman akan otomatis
menampilkan placeholder ikon + teks (tidak akan menampilkan ikon "broken image").

Rekomendasi ukuran:
- couple.jpg  : persegi/potrait, minimal 500x650 px
- adjie.jpg / galuh.jpg : rasio 3:4, minimal 600x800 px
- journey.jpg / closing.jpg : rasio 4:5, minimal 640x800 px

Kompres foto (misal lewat squoosh.app atau tinypng.com) agar halaman tetap ringan,
idealnya di bawah 300–500 KB per foto.
